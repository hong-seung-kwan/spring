문자열 객체

  • aBC에 a가 포함되었는지? [[ ${ #strings.contains('aBC','a') } ]]
  • aBC가 b로 시작되었는지 : [[${#strings.startsWith('aBC', 'b')}]]
  • aBC를 인덱스 2부터3전까지 자르기 : [[${#strings.substring('aBC', 2, 3)}]]
  • aBC에서 a를b로바꾸기 : [[${#strings.replace('aBC', 'a', 'b')}]]
  • aBC를 대문자로 변환 : [[${#strings.toUpperCase('aBC')}]]
  • aBC를 소문자로 변환 : [[${#strings.toLowerCase('aBC')}]]

숫자객체

  • 자리수만큼 0으로 채우기: [[ ${ #numbers.formatInteger(1, 5) } ]]
  • 패턴을 지정하여 출력 : [[${#numbers.formatInteger(100000,3,'POINT')}]]
  • 화폐단위로 출력 : [[${#numbers.formatCurrency(100)}]]
  • 퍼센트단위로 출력 : [[${#numbers.formatPercent(0.2,2,0)}]]

날짜 객체

  • 현재 시간: [[ ${ date } ]]
  • 날짜를 특정패턴으로 출력: [[ ${#temporals.format(date,'yyyy/MM/dd' )}]]
  • 시간을 특정패턴으로 출력: [[ ${#temporals.format(date,'hh시mm분ss초' )}]]
  • 시간만 출력: [[ ${#temporals.format(date,'hh시' )}]]
  • 연도 : [[${#temporals.year(date)}]]
  • 월 : [[${#temporals.month(date)}]]
  • 일 : [[${#temporals.day(date)}]]
